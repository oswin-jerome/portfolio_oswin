# Billocity

### Tech Used

- Laravel
- MySQL

![alt text](/works/billocity.jpg)

This is a retail invoicing & stock management system built with laravel & mysql as a backend and blade with bootstrap as frontend tools.

### Features

- Stock Management
- Invoicing
- Payment Tracking
- EMI
- Reports
